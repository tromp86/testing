Що таке об'єкт в JavaScript?

Об'єкт - це змінна, яка може містити різноманітні значення та методи. Об'єкти в JavaScript можуть бути створені як літеральні об'єкти, використовуючи фігурні дужки {}, або як об'єкти класу.

Які основні методи та властивості у об'єктів?

Об'єкти мають ряд методів та властивостей. Деякі з найбільш використовуваних методів включають: Object.keys(obj) - повертає масив ключів об'єкта; Object.values(obj) - повертає масив значень об'єкта; Object.assign(target, source) - копіює значення властивостей з одного або більше об'єктів у цільовий об'єкт; Object.freeze(obj) - забороняє змінювати властивості об'єкта; Object.create(proto) - створює новий об'єкт з заданим прототипом. Основні властивості об'єктів включають: obj.length - кількість властивостей в об'єкті; obj.constructor - функція-конструктор, яка була використана для створення об'єкта; obj.prototype - прототип об'єкта.

Що таке this в JavaScript?

this в JavaScript відноситься до поточного об'єкту, який викликав поточну функцію. this дозволяє звернутися до властивостей та методів поточного об'єкту в контексті поточної функції.

Як визначити значення this в JavaScript?

Визначення значення this в JavaScript залежить від контексту, у якому була викликана поточна функція. Якщо функція викликана з методу об'єкта, то значення this буде посилатися на цей об'єкт. Якщо функція викликана без контексту, то значення this буде посилатися на глобальний об'єкт window. Можна також використовувати методи call() та apply() для явного вказання значення this.

Які є різниці між методами call() та apply() в JavaScript?

Методи call() та apply() в JavaScript дозволяють викликати функцію з явно вказаним значенням this. Основна різниця між ними полягає в способі передачі аргументів до функції. У методі call() аргументи передаються окремими параметрами, тоді як у методі apply() аргументи передаються в масиві.

Як можна змінити значення this в JavaScript?

Можна змінити значення this в JavaScript за допомогою методів call(), apply() або bind(). Ці методи дозволяють явно вказати значення this для функції. Також можна використовувати стрілкові функції, які не мають свого контексту this та приймають його з контексту, у якому вони були оголошені.

як обєкт привести до примітиву?

У JavaScript об'єкти можна перетворити на примітиви за допомогою явного або неявного приведення типу. Неявне приведення типу відбувається при використанні об'єкта в контексті, де очікується примітив. Наприклад, при додаванні об'єкта до рядка, JavaScript автоматично викликає метод toString() об'єкта, щоб перетворити його на рядок.

Явне приведення типу до примітиву можна здійснити за допомогою вбудованих методів, таких як toString(), valueOf() або Symbol.toPrimitive().

Також можна перевести об'єкт на примітив, використовуючи метод valueOf(). Цей метод повертає найбільш базовий примітивний варіант об'єкта.

що таке поле обєкту?

Поле об'єкту - це змінна, яка міститься всередині об'єкту та зберігає значення. Це можуть бути будь-які дані, такі як числа, рядки, булеві значення, масиви, інші об'єкти або навіть функції. Поля об'єкту зазвичай використовуються для зберігання даних, які пов'язані з цим об'єктом та які можуть бути використані в процесі його використання.

Свойство об'єкту - це змінна, що міститься всередині об'єкту та зберігає значення, а також може мати пов'язані з ним методи. Свойства об'єкту визначаються за допомогою ключа (імені), який вказується в об'єкті, а потім пов'язується зі значенням.
