TypeScript

- це статична мова програмування, яка дозволяє визначати типи даних іншим чином, ніж у звичайному JavaScript, що допомагає запобігати багатьом помилкам і полегшує розробку більших проектів.

Типи даних

TypeScript дозволяє визначати типи даних, що використовуються в програмі, такі як number, string, boolean і т.д. Крім того, TypeScript також дозволяє визначати користувацькі типи даних.

Класи

Класи в TypeScript дуже схожі на класи в звичайному JavaScript. Однак, TypeScript дозволяє визначати поля та методи класу з їх типами, що дозволяє запобігти багатьом помилкам.

Інтерфейси

Інтерфейси в TypeScript використовуються для визначення типів об'єктів. Інтерфейс може містити визначення полів, методів та їх типів.

Дженеріки

Дженеріки в TypeScript дозволяють створювати функції та класи, які можуть працювати з будь-яким типом даних.

Декоратори

Декоратори в TypeScript використовуються для додавання функцій до класів, методів та інших елементів програми.

Array: масиви

Тип Array у TypeScript є впорядкованим списком елементів певного типу. Масиви можуть бути створені за допомогою наступного синтаксису:

let list: number[] = [1, 2, 3];

let list: Array = [1, 2, 3];

Кортежи (Tuples)

є масиви з фіксованим набором елементів і типами. Також кортежі можуть бути використані для опису значень, що повертаються функціями.

let tuple: [string, number]; tuple = ["hello", 10]; // ОК tuple = [10, "hello"]; // помикла)

Enum: перерахування

є набором числових значень, які можуть бути названі. Enum дозволяє створювати більш зчитуваний і зрозумілий код.

enum Color {Red, Green, Blue}; let c: Color = Color.Green; console.log(c); // 1

Any: довільний тип

є типом даних, який може бути використаний для представлення значень різних типів. Any дозволяє уникнути помилок типізації, але може призвести до помилок під час виконання програми.

let notSure: any = 4; notSure = "maybe a string instead"; notSure = false;

Symbol

є унікальним значенням, яке може бути використане як ключ об'єкта. Вони часто використовуються для створення ідентифікаторів властивостей об'єктів.

let sym1 = Symbol(); let sym2 = Symbol("key"); // Символ з описом "key"

Null та undefined

відповідають значенням null і undefined JavaScript. Вони можуть бути використані як окремі типи даних.

let u: undefined = undefined; let n: null = null;

Never

є типом даних, який не має жодних значень. Він використовується як тип функцій, що повертаються, які генерують або повертають помилку. Наприклад, функція, яка завжди викликає помилку:

function error(message: string): never {throw new Error(message);}
