Щоб викласти проект на Github, слід дотримуватися наступних кроків:

Ініціалізуйте репозиторій, використовуючи git. У вас повинен бути встановлений git на вашому комп'ютері, а потім запустіть наступні команди в командному рядку:

cd /шлях/до/вашого/проекту

git init

git add .

git commit -m "Initial commit"

git remote add origin https://github.com/.git

git push -u origin master

Після виконання цих команд ваш проект буде завантажено на GitHub.
